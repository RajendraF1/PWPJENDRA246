Our Features

Halaman Registrasi

  • Form HTML untuk registrasi pengguna.
  • Data yang dimasukkan akan disimpan ke database dengan hash password.

Halaman Login

  • Form HTML untuk login pengguna.
  • Jika login berhasil, simpan informasi pengguna ke session.

Halaman Dashboard (Hanya untuk Pengguna Login)

  • Menampilkan daftar semua pengguna.
  • Memiliki tombol untuk menambahkan, mengedit, dan menghapus pengguna.

Fitur CRUD

  • Tambah Pengguna: Form untuk menambahkan pengguna baru.
  • Edit Pengguna: Form untuk mengedit data pengguna berdasarkan ID.
  • Hapus Pengguna: Tombol untuk menghapus pengguna.

Halaman Logout

  • Tombol untuk logout pengguna dan menghapus session.

Model Database

  • id.
  • username.
  • role.
  • email.
  • password_hash.

About Us

Selamat datang di aplikasi web saya, yang merupakan hasil dari tugas responsi Mata Kuliah Pemrograman Web Praktik. Aplikasi ini dibangun menggunakan Flask sebagai framework backend, yang memanfaatkan REST API untuk komunikasi antar client dan server. Data pengguna disimpan dan dikelola menggunakan MySQL sebagai sistem manajemen basis data. Untuk meningkatkan pengalaman pengguna, tampilan aplikasi didesain dengan gaya yang responsif dan modern menggunakan Tailwind CSS CDN.

Saya juga menerapkan session untuk mengelola status pengguna dan autentikasi, memastikan pengalaman yang lebih aman dan personal. Aplikasi ini dirancang untuk memberikan pemahaman yang mendalam mengenai penerapan teknologi web terkini dalam pengembangan aplikasi berbasis RESTful dan interaktif.
